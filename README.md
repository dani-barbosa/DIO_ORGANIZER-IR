# DIO_ORGANIZER-IR
Trata-se de um organizador de informações para preenchimento da Declaração de IR
O objetivo proposto foi criar uma ferramenta no Excel que permita agrupar de forma organizada as principais informações para o preenchimento de uma declaração de imposto de renda. A proposta foi construir o instrumento que permita que o usuário preencha os campos indicados para serem alimentados, enquanto a ferramenta irá auxilia-lo no controle dos dados inseridos, separando as categorias por abas/ menu de navegação.
